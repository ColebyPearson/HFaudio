# HuggingFace Audio Transformers Course

Working through the [HuggingFace Audio Course](https://huggingface.co/learn/audio-course/chapter0/introduction) to earn the course certificate.

## Course Units

| Unit | Topic | Type |
|------|-------|------|
| 1 | Working with Audio Data | Theory + Code |
| 2 | Audio Applications with Pipelines | Theory + Code |
| 3 | Transformer Architectures for Audio | Theory |
| 4 | Music Genre Classifier | **Hands-on Exercise** |
| 5 | Speech Recognition (ASR) | **Hands-on Exercise** |
| 6 | Text-to-Speech (TTS) | **Hands-on Exercise** |
| 7 | Putting It All Together | **Hands-on Exercise** |

## Certification

| Type | Requirement |
|------|-------------|
| Certificate of Completion | 3 of 4 hands-on exercises |
| Certificate of Honors | All 4 hands-on exercises |

## Setup

```bash
pip install -r requirements.txt
```

Notebooks are designed to run on Google Colab with free GPU access.
